# Side-Scrolling Dragon 🐉

ターミナル上で動く横スクロール型の障害物回避ゲームです。  
ドラゴンを操作して飛んでくる柱状の障害物をジャンプでかわし、ハイスコアを目指しましょう。

## 必要環境

- [Rust](https://www.rust-lang.org/) 1.85 以上（Edition 2024 対応）

## インストールと起動

```bash
# リポジトリをクローン
git clone https://github.com/MenEnger/side-scrolling-dragon.git
cd side-scrolling-dragon

# ビルドして起動
cargo run --release
```

## 操作方法

| キー | 動作 |
|------|------|
| `Space` | ジャンプ（最大 2 段ジャンプ可能） |
| `R` | ゲームをリスタート |
| `Q` / `Esc` | ゲームを終了 |

## ゲームルール

- 右から流れてくる柱状の障害物（`#`）をジャンプで回避します。
- ドラゴン（`<(o )==>`）が障害物に衝突するとゲームオーバーです。
- 障害物を 1 本通り過ぎるごとにスコアが 1 加算されます。
- スコアが上がるにつれてスクロール速度が増していきます。
- ジャンプは 1 度の着地前に最大 **2 回** まで可能です（2 段ジャンプ）。

## ドラゴンのスプライト

ジャンプ状態に応じてスプライトが変化します。

| 状態 | スプライト |
|------|-----------|
| 地上 | `<(o )==>` |
| 1 段目ジャンプ | `<(o )~>>` |
| 2 段目ジャンプ | `<(O )**>` （炎が橙色に点灯） |
